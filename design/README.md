# Run the NexusMed prototype

This folder contains a standalone browser prototype. It does not call external
AI providers and it must not embed `OPENAI_API_KEY` or any other server secret.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
